Sistema de Inventario – Caso Pepsi (Planta Embotelladora)

Manejo de botellas vacías, tapas y jarabe concentrado para asegurar producción continua sin quiebres de stock. Esta página explica el sistema de control de inventario (EOQ, Stock de Seguridad, Punto de Reorden) y presenta un caso práctico con datos simulados de una planta embotelladora Pepsi.

Proyecto académico · Valores aproximados · Uso en clase

1. Contexto operativo

• Una planta embotelladora Pepsi necesita flujo constante de insumos: botellas vacías, tapas, etiquetas y jarabe.

• Si falta un insumo por 1 día, la línea de llenado se detiene y se pierden ventas inmediatas.

• Para evitar eso, se maneja stock de seguridad y se aplican fórmulas de inventario.

• Meta: tener producto listo sin saturar bodega, porque inventario excesivo = dinero inmovilizado.

• El control gira alrededor de: EOQ, Stock de Seguridad y Punto de Reorden.

2. Sistema de control de inventario y fórmulas clave

2.1 EOQ (Cantidad Económica de Pedido)

Determina cuántas unidades pedir cada vez para minimizar el costo total de pedir vs. almacenar inventario.

EOQ = √((2 × D × Co) / Cc)
D = Demanda anual (unid/año)
Co = Costo de hacer un pedido (Q por pedido)
Cc = Costo anual de mantener 1 unidad (Q por unidad·año)

2.2 Stock de Seguridad (Safety Stock)

Inventario colchón usado cuando la demanda sube más de lo normal o cuando el proveedor se atrasa.

SS = Z × σd × √L
Z = Nivel de servicio deseado (ej. 1.65 ≈ 95%)
σd = Variación (desviación estándar) de la demanda diaria
L = Lead time en días (lo que tarda el proveedor)

2.3 Punto de Reorden (ROP)

Nivel de inventario al que se dispara el siguiente pedido.

ROP = d × L + SS
d = Demanda diaria promedio
L = Lead time en días
SS = Stock de Seguridad

Cuando el inventario llega al ROP, automáticamente se genera un nuevo pedido.

4. Conclusiones rápidas
  • EOQ = cuánto pedir cada vez.
  • ROP = cuándo volver a pedir.
  • Stock de Seguridad = colchón de emergencia.
  • Mucho inventario = dinero parado en bodega.
  • Poco inventario = paro inmediato de producción.
3. Caso práctico: Planta Embotelladora Pepsi Guatemala
Supongamos que la planta de Pepsi compra botellas plásticas vacías a un proveedor externo. Para evitar parar la línea, necesitamos saber: cuánto pedir (EOQ), cuándo pedir (ROP), y cuánto guardar como colchón (Stock de Seguridad).

Nota: asumimos ~300 días de operación al año.

Datos base:
Dato Valor
Demanda anual (D) 2,400,000 botellas/año
Costo por pedido (Co) Q 150 por pedido
Costo de mantener (Cc) Q 0.05 por botella·año
Demanda diaria promedio (d) 8,000 botellas/día
Desviación estándar diaria (σd) 1,200 botellas
Lead time proveedor (L) 4 días
Nivel de servicio deseado (Z) 1.65 (≈95%)
Resultados calculados:

Paso 1. EOQ

EOQ = √((2 × 2,400,000 × 150) / 0.05)

EOQ ≈ 120,000 botellas por pedido

Interpretación: conviene pedir ~120,000 botellas cada vez para equilibrar costo de ordenar vs costo de almacenar.

Paso 2. Stock de Seguridad

SS = 1.65 × 1,200 × √4

SS ≈ 3,960 botellas extra

Ese es el colchón para cubrir picos de demanda o retrasos del camión proveedor.

Paso 3. Punto de Reorden (ROP)

ROP = (8,000 bot/día × 4 días) + 3,960

= 32,000 + 3,960

ROP ≈ 35,960 botellas

Cuando el inventario baja a ~35,960 botellas, se debe lanzar el siguiente pedido.

5. Actividad en clase
  1. Cambia el lead time a 6 días.
    • Recalcula Stock de Seguridad y ROP.
    • ¿El ROP sube o baja? Explica.
  2. Cambia la demanda diaria a 6,500 botellas/día.
    • ¿Cambia el EOQ?
    • ¿Cambia el ROP? ¿Por qué?
  3. Propón una política de compra para Pepsi:
    • ¿Cuántas veces al mes harías pedido?
    • ¿Cuántas botellas cada vez?
6. Conclusiones
  • El costo de inventario no es sólo “tener cajas en bodega”. También significa espacio físico, manejo, riesgo de daño/caducidad y dinero inmovilizado.
  • EOQ define cuánto pedir por lote.
  • El Punto de Reorden (ROP) define cuándo pedir otra vez.
  • El Stock de Seguridad es tu plan B para que la línea de Pepsi no se quede sin botellas.
  • Si fallan estos cálculos → paro de producción inmediato, pérdida de ventas en el mismo día.
